Find IBD Shared Haplotypes Rapidly

About

FISHR is a project from the Keller Lab at the University of Colorado, Boulder that focuses on rapidly finding identity by descent (IBD) shared haplotypes in a set of genetic data. The set of programs are written in C++, and are all open source.

FISHR process overview

The FISHR program suite employs the following processes:

  • Run genetic data through a customized version of Germline
  • Check resulting shared haplotypes for consolidation opportunities
  • Perform statistical analysis on shared haplotypes that are likely IBD
  • Perform statistical analysis over all shared haplotypes in the data set
  • Filter data set on certain criteria
  • Procude several formats of output detailing the shared haplotypes that passed the systems filters

A project tailored to your computing needs

We have written a highly optimized version of FISHR that can run smoothly in low-RAM computing environments as well.

Software

The FISHR project is a suite of programs designed to find IBD shared haplotypes as rapidly as possible. It consists of the following programs:

  • Fishr (Error Finder) - The main statistical analysis engine
  • Fishr Low RAM - A version of Fishr optimized for low-RAM environments
  • GL Parameter Finder - A program that helps calculate empirical inputs for Fishr
  • Germline - A customized version of the software from Columbia that prepares inputs for Fishr
